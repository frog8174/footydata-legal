FootyData Uploader（下稱「本服務」）為頻道擁有者自用之自動化內容上傳程式，僅用於管理擁有者本人之 YouTube 頻道。

YouTube API Services

本服務使用 YouTube API Services 存取資料。使用本服務即表示您了解並同意 YouTube 服務條款（YouTube Terms of Service）。Google 如何處理資料請見 Google 隱私權政策（Google Privacy Policy）。

我們存取的資料 / Data We Access

僅存取本頻道擁有者授權之資料：上傳擁有者自製影片至其自有頻道（youtube.upload）、讀取其自有頻道與影片之資訊與成效數據（youtube.readonly、yt-analytics.readonly）。不蒐集任何第三方使用者個資。

使用與分享 / Use & Sharing

資料僅用於上傳原創影片與優化自有頻道內容；不出售、不轉讓、不與任何第三方分享。本服務對 Google user data 之使用遵循 Google API Services User Data Policy（含 Limited Use 規定）。

資料刪除政策 / Data Deletion & Revocation

使用者可隨時至 Google 帳戶安全性設定（myaccount.google.com/permissions） 撤銷本應用程式之存取權。一旦撤銷，本服務即停止一切 API 存取，並刪除所儲存之 OAuth 授權憑證（refresh token）及相關快取資料。如需協助刪除，請來信 chenz8174@gmail.com，我們將於合理期間內處理。

安全 / Security

授權憑證儲存於受限的伺服器環境，僅限維運人員存取。
