FootyData Uploader — 隱私權政策 / Privacy Policy

最後更新 Last updated: 2026-06-15

本工具（FootyData Uploader，下稱「本服務」）為頻道擁有者自用之自動化內容上傳程式，僅用於管理擁有者本人之 YouTube 頻道。This application is a private, owner-operated automation tool used solely to manage the operator's own YouTube channel.

我們存取的資料 / Data We Access

透過 Google／YouTube API，僅存取本頻道擁有者授權之資料：將擁有者自製影片上傳至其自有頻道（youtube.upload）、讀取其自有頻道與影片之資訊與成效數據（youtube.readonly、yt-analytics.readonly）。

使用方式 / How We Use It

僅用於上傳本服務自製之原創影片，以及分析自有影片成效以優化內容。我們不會蒐集、儲存或處理任何第三方使用者之個人資料。

資料分享 / Data Sharing

我們不會將透過 Google API 取得的資料出售、轉讓或分享給任何第三方。

資料儲存與安全 / Storage & Security

授權憑證儲存於受限的伺服器環境，僅限維運人員存取。

Google API 合規 / Compliance

本服務使用 Google API 取得的資訊皆遵循 Google API Services User Data Policy（含 Limited Use 規定）。
